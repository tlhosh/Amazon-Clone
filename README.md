# Amazon Clone

A fully functional e-commerce landing page inspired by Amazon, featuring product browsing, a dynamic shopping cart, and a checkout flow. The project demonstrates front-end development skills, state management, and basic e-commerce logic.

## Features

* **Landing Page**

  * Product listings similar to an online store layout
  * Responsive design for mobile and desktop

* **Shopping Cart**

  * Add/remove items from cart
  * Update item quantities
  * Real-time cart total calculation

* **Checkout Flow**

  * Order summary page
  * Basic checkout UI (can be extended with payment integration)
  * Clear cart after checkout (if implemented)

* **Product Interaction**

  * View product details
  * Add products to cart from listing page

## Tech Stack (example)

* HTML5
* CSS3 / Tailwind CSS / Bootstrap
* JavaScript (Vanilla or React)
* LocalStorage (for cart persistence, if used)

## How It Works

1. Products are loaded into the landing page (static or JSON data).
2. Users click Add to Cart to store items in a cart array/state.
3. Cart updates dynamically and calculates totals.
4. Checkout page summarizes the order before final confirmation.
